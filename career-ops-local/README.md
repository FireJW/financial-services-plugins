# Career Ops Local

`career-ops-local` is a private, local-first bridge for job-search operations.

It is designed to:

- keep resumes, trackers, and portal data outside this repo
- reuse this repo's command/skill/runtime patterns
- prefer local logic when it is already as good or better
- keep the external `career-ops` checkout optional

See:

- `docs/runtime/career-ops-local-operator-setup.md`
- `docs/runtime/career-ops-local-capability-map.md`
