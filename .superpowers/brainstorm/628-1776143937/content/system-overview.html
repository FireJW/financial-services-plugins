3 个内容领域 × 视觉风格矩阵

宏观 / 财经 / 地缘

紧张、权威、信息密度高

文章风格：短句快评、信号解读、情景推演

视觉方向：

· 有现场 → 纪实摄影（A）
· 无现场 → 抽象隐喻（C）

色调：暗调、去饱和暖色、紧张感

科技动态

前沿、清晰、有未来感

文章风格：产品解读、技术趋势、行业分析

视觉方向：

· 有产品实体 → 产品特写/场景（新方向 D）
· 纯概念/趋势 → 科技抽象（新方向 E）

色调：冷色调、深蓝/紫、干净、高对比

个人成长

温暖、有质感、不鸡汤

文章风格：经验复盘、思维模型、读书笔记

视觉方向：

· 有具体场景 → 生活质感摄影（新方向 F）
· 纯思维/概念 → 极简隐喻（新方向 G）

色调：暖色、柔光、自然、有呼吸感
