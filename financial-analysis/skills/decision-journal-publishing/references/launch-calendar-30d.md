# 30-Day Launch Calendar

## Goal

Use the first 30 days to prove three things:

1. the audience understands the evidence-layered format
2. the lite template captures qualified demand
3. the core template pack can close real buyers

## Posting System

Default weekly rhythm:

- WeChat: 2 posts per week
- Toutiao: 3 fast cards per week
- Xiaohongshu: optional 0-1 reuse slice per week
- X: deferred

## CTA Progression

### Days 1-10

Primary CTA:

- 领取 Lite 模板

### Days 11-17

Primary CTA:

- 进入模板包等候名单

### Days 18-30

Primary CTA:

- 购买事件驱动研判模板包 v1

## WeChat Core Posts

The first month should keep WeChat to eight core posts:

1. 一条突发消息，为什么不能直接变成结论
2. 两家媒体说法相反时，先看哪一个信号
3. 一次政策冲击里，真正该先看哪 3 个变量
4. 一次政策冲击，哪些信号会让我升级判断
5. 事件驱动研判模板包 v1 正式说明
6. 模板包里的 AI 工作流到底怎么用
7. 一套模板怎么帮你省 2 小时
8. 第一个月总结：哪些内容真正转化

## Notes

1. Every WeChat post should end with one CTA, not three.
2. Quiet days are mandatory; use them for buyer support, pack refinement, and review.
3. Toutiao cards should feel complete on their own, then point upward into WeChat.
4. Public scorekeeping replaces prophecy-style trust building.
5. Day 18 is a target launch date, not a ceremonial deadline. If qualified demand is still weak, extend the waitlist phase and publish more proof content before opening payment.
6. If a live event already offers a stronger public hook than a planned method post, swap the wrapper but keep the same underlying template and CTA.
