# Platform System

## Phase-1 Default

For the first 30 days, treat the system as a two-platform core:

1. WeChat Official Account
2. Toutiao

Add Xiaohongshu only as light reuse when a post already has checklist or flowchart
value.

Defer X until the core pack has already sold and the production loop is stable.

## Role Split

| Platform | Main Job | Best Formats | Conversion Role |
|---|---|---|---|
| WeChat Official Account | trust, archive, conversion | long-form template demos, case reviews, update notes | primary |
| Toutiao | cold-start traffic, angle testing, topic radar | fast cards, conflict summaries, anti-hype explainers | indirect |
| Xiaohongshu | optional save-worthy reuse | checklists, flowcharts, template slices | indirect |
| X | later English sidecar | short summaries, source notes | deferred |

## WeChat

Use for:

1. complete template walkthroughs
2. worked examples
3. weekly review logs
4. product update notes
5. lead magnet and template-pack conversion

Structure every long-form post:

1. bottom line
2. evidence layer
3. what is still unclear
4. trigger board
5. what would change my view
6. CTA
7. mandatory footer

Do not use WeChat as a dumping ground for daily hot-news fragments.

Benchmark implication:

1. WeChat can still produce 10W+ finance or business articles
2. but recommendation-driven spikes are not always precise traffic
3. therefore WeChat should carry both public-hook explainers and conversion-safe
   core posts

## Toutiao

Use for:

1. event-driven reach
2. title and angle testing
3. short-form trust signals
4. retail-safe acquisition into WeChat

Structure every card:

1. one-line takeaway
2. confirmed vs unconfirmed
3. one conflict point
4. next thing to watch
5. one-line boundary statement when the topic is sensitive
6. WeChat CTA

Toutiao should feed the funnel, not become the product definition.

Benchmark implication:

1. Toutiao currently rewards deep analytical text more than many operators assume
2. winning posts usually start from consequence, conflict, or an affected group
3. framework language belongs inside the post, not in the headline shell

## Xiaohongshu

Use only when the content naturally becomes:

1. a checklist
2. a flowchart
3. a save-worthy template fragment
4. a retail-safe "how to read this event" note

Do not force Xiaohongshu into the first-month cadence.

## X

Use later for:

1. English mirror posts
2. source discovery
3. reputation and network

Do not assign first-quarter revenue or production targets to X.

## Distribution Principle

In phase 1, one evidence pack should generate:

1. one WeChat long-form post
2. one Toutiao fast card
3. zero or one Xiaohongshu slice only if it is naturally save-worthy

Packaging rule:

1. the WeChat wrapper may lead with the most decision-useful angle
2. the Toutiao wrapper should usually lead with the most public-facing
   consequence angle

Do not create four separate research processes.

## Platform Admission Rules

Only add a new platform when all three are true:

1. WeChat is publishing steadily
2. Toutiao is already delivering qualified leads
3. the template pack sales loop is functioning

If any of those are false, fix the core system first.

## CTA Split

| Platform | CTA |
|---|---|
| WeChat | get lite template / join template-pack waitlist / buy template pack |
| Toutiao | read the full framework on WeChat |
| Xiaohongshu | save this slice, then get the lite template |
| X | follow for English evidence summaries |
