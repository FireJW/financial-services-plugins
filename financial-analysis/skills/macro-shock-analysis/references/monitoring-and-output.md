## 1. Early-Warning Signals

These are the signals most likely to tell you whether a shock is embedding.

### Inflation process

- core inflation breadth
- services inflation
- PPI to CPI pass-through
- market breakevens
- survey inflation expectations

### Labor

- wage settlements
- hourly earnings
- labor shortages in exposed sectors

### Real economy

- inventories
- PMIs
- industrial output
- freight volumes
- import compression

### External balance and FX

- importer FX weakness
- current-account pressure
- reserve use
- sovereign spreads

### Policy

- official language shifting from "temporary" to "persistent"
- subsidy or tax interventions
- reserve releases
- emergency liquidity measures

### Energy-specific

- Brent and prompt spreads
- Dubai or Oman crude spreads
- European gas benchmarks
- Asian LNG benchmarks
- U.S. Henry Hub versus global gas benchmarks
- tanker rates and rerouting times
- refinery cracks
- official reserve-release or export-control signals

## 3. Reaction Function Template

For each policy actor, write:

1. current tolerance
2. trigger for action
3. likely first tool
4. likely limit

Minimum actors:

- central bank
- fiscal authority
- regulatory / administrative authority

## 5. Final Note Structure

Prefer this order:

1. `Shock in one line`
2. `Why it matters now`
3. `What is direct vs second-round`
4. `Our persistence judgment`
5. `Scenario matrix`
6. `Policy reaction function`
7. `Market implications`
8. `Indicator board`
9. `What we are watching next`
10. `What would change our mind`

When the event depends on policy retreat risk, add:

11. `Policy pressure / pivot risk`

## 6. Policy Pressure Overlay

Use this section when:

- the event depends on Trump backing off, delaying, narrowing, or softening a
  policy
- the user explicitly asks about `TACO`, `pivot risk`, `retreat odds`, or a
  similar political-market feedback loop

Minimum output:

| Section | Required content |
|---------|------------------|
| Pressure composite | Component list and likely sign |
| Read-through | What a higher reading means for policy retreat odds |
| Why it may matter now | Which components are currently aligned or not |
| False positives | What could make the composite misleading |
| What changes the view | What would raise or reduce retreat odds |

## 7. Standardized Output Skeleton

Use this output shape by default.

### A. One-line judgment

- `One-line judgment:` one sentence only

Example:

- `One-line judgment: This is a large energy shock, but still a contained one unless physical disruption lasts beyond the current risk window.`

### B. Core table

| Section | Required content |
|---------|------------------|
| Shock in one line | What changed, first channel, why it matters |
| Current state | Confirmed facts vs unresolved facts |
| Physical disruption vs risk premium | What is real outage vs repricing only |
| Transmission map | Direct, indirect, second-round |
| Persistence judgment | Why it likely fades or embeds |
| Scenario matrix | Three scenarios minimum |
| Reaction function | Central bank, fiscal, regulatory |
| Policy pressure / pivot risk | Only when retreat probability matters |
| Market implications | By asset bucket or benchmark |
| Indicator board | What to monitor next |
| What changes the view | Upgrade and downgrade triggers |

### C. Bias table

Always include a short comparison table near the end.

| Bias | Conditions that strengthen this bias | Conditions that weaken this bias |
|------|--------------------------------------|----------------------------------|
| Bullish / risk-up in price |  |  |
| Bearish / fade risk premium |  |  |

If the task is not about price direction, rename the left column to the relevant
view, for example `More inflationary` vs `Less inflationary`.

### C1. Confidence markers

Always state both:

| Confidence type | Meaning |
|-----------------|---------|
| Evidence confidence | How strong the factual base is |
| Scenario confidence | How strong the forward-looking judgment is |

Preferred labels:

- `High`
- `Medium`
- `Low`

### E. Preliminary mode variant

If the evidence base is incomplete, replace hard scenario probabilities with:

- `high confidence`
- `medium confidence`
- `low confidence`

And add:

- `Unresolved facts that matter most`

## 8. Quality Checks

Before finishing, make sure the note does all of this:

1. explains the causal chain
2. distinguishes observed facts from inference
3. does not overclaim policy power
4. does not rely on one baseline only
5. states why the shock may fade or embed
6. gives concrete watch items instead of vague caution
7. separates what is already priced from what is still a scenario extension
8. if using a policy-pressure composite, treats it as an overlay rather than
   direct proof of a pivot
