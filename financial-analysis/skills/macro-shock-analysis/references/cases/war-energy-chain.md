# Case: War, Energy, And Beneficiary Chain

Use this case when the user asks how war escalation, shipping disruption,
Hormuz risk, sanctions, or negotiation breakdown affects oil, LNG, inflation,
and related stocks.

## Default Routing

1. If facts are moving, start with
   [autoresearch-info-index](../../autoresearch-info-index/SKILL.md).
2. Only after the fact pack is clean, run macro shock analysis.
3. If the user wants stocks, map the macro transmission into direct,
   second-order, and false-beneficiary buckets.

## Core Split

Always separate:

- physical disruption vs risk premium
- oil vs LNG vs regional gas benchmarks
- first-round price effect vs second-round margin effect
- macro regime risk vs short-lived headline trade

## Stock Judgment Discipline

Never stop at `oil up so energy stocks up`.

Classify each company first:

1. resource owner or upstream producer
2. service or equipment supplier
3. energy-security or domestic-substitution beneficiary
4. downstream processor or cost taker

## Standard Output

Prefer:

- one-line judgment
- fact board
- core comparison table
- transmission chain
- horizon split: `0-72h`, `1-4w`, `1-3m`
- `bias stronger if`
- `bias weaker if`
- what would invalidate the view

## What To Watch

- benchmark choice: Brent, TTF, JKM, Henry Hub, tanker or freight proxies
- persistence signals: rerouting, inventories, spare capacity, formal closure,
  export interruptions
- policy response: subsidies, releases, diplomacy, military de-escalation
- market read-through: breakevens, front-end rates, importer FX, shipping

## When To Stay Preliminary

Stay in preliminary mode when:

- the disruption is not yet confirmed physically
- military location data is only public-indication quality
- there is still active contradiction between official and social reporting
